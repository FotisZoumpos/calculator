# Επιστημονικό Κομπιουτεράκι (Scientific Calculator)

Αυτή η εφαρμογή υλοποιεί ένα επιστημονικό κομπιουτεράκι σε δύο μορφές:
- **Command Line Interface (CLI)**
- **Graphical User Interface (GUI)**

## Ονομασία Αρχείων

Τα αρχεία των κώδικων έχουν την εξής μορφή:
xxxxxx_Code_CN6000
Όπου `xxxxxx` είναι ο αριθμός μητρώου UEL του φοιτητή.

### Σημαντική Παρατήρηση για MATLAB

Επειδή το MATLAB απαιτεί τα ονόματα αρχείων να ξεκινούν με γράμμα, έχει γίνει η εξής τροποποίηση:

- Το αρχείο CLI ξεκινά με το γράμμα **`a`** , (a_2473536_Code_CN6000_CLI)
- Το αρχείο GUI ξεκινά με το γράμμα **`b`** , (b_2473536_Code_CN6000_UI_App_Designer)

#### Εκτέλεση GUI στο MATLAB

Για να εκτελέσετε το αρχείο GUI:

1. Ανοίξτε το MATLAB.
2. Στο Command Window, πληκτρολογήστε: appdesigner και πατήστε "Enter".
3. Από το παράθυρο του App Designer, επιλέξτε **Open...**.
4. Δώστε τη διαδρομή του αρχείου `.mlapp` για να το ανοίξετε και να το εκτελέσετε.
